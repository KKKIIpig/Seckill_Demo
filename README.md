### 秒杀业务分析

秒杀业务主要是在商品增加调整库存，用户对库存进行购买秒杀的行为

主要核心是对库存进行处理

这里使用的是 MySQL（事务+行锁） ，现在 redis 也能很好的支持事务，尤其适合在高并发情况下完成。
